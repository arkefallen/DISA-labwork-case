Pada Proyek Akhir Matakuliah Praktikum Pemrograman Web 1 ini saya membuat "Aplikasi Daftar Penghuni Kos Aman Sentosa". Ide dari pembuatan aplikasi ini adalah saat saya memikirkan permasalahan yang dialami pemilik kos yaitu tidak bisa mengingat semua identitas penghuni kos yang ada. Karena permasalahan ini, saya mencoba mengimplementasikan sebuah solusi dengan menciptakan aplikasi ini.

Aplikasi ini saya buat kurang lebih menggunakan Konsep CRUD. Apa itu CRUD ?
CRUD singkatan dari Create, Read, Update, dan Delete yaitu sebuah fungsi2 utama yang harus ada di dalam aplikasi database yang sudah kita buat. Nah di aplikasi yang saya buat, bisa menerapkan fungsi2 tersebut, antara lain aplikasi ini bisa menambah penghuni kos baru, bisa melihat data siapa saja yang sekarang sedang menyewa kamar kos, bisa mengubah data penghuni kos, dan bisa menghapus data penghuni saat ini apabila ada yang sudah tidak menempati kos lagi.


Selama 1 semester belajar Praktikum Pemrograman Web 1, saya sudah belajar HTML, CSS, Bootstrap, Javascript, PHP, dan Web Design. Dalam proyek akhir yang saya buat, alhamdulillah saya dapat menerapkan materi-materi tersebut. 

1. Pertama, saya melakukan desain website/ UI Design terlebih dahulu dari materi yang telah diajarkan yaitu Web Design.

5. Terakhir, saya menggunakan PHP agar bisa mengoneksikan antara data penghuni yang ada di database dengan website yang sudah dibuat agar bisa melakukan fungsi-fungsi CRUD.
